如何获取
从浏览器 DevTools → Application → Cookies 复制，或用 EditThisCookie 等扩展导出为 JSON

点击选择 cookies.json 文件

方法1（推荐）：用 EditThisCookie 扩展
1. Chrome 安装 EditThisCookie 扩展
2. 打开并登录 agent.minimaxi.com
3. 点击扩展图标 → 导出为 JSON
4. 粘贴到「粘贴 JSON」标签页

方法2：浏览器 DevTools
1. 登录后按 F12 → Application → Cookies → agent.minimaxi.com
2. 全选所有行，复制为 JSON（或用脚本：document.cookie）

方法3：本地运行收集脚本
在 Windows 本地执行 node collect-cookies.js，
会弹出浏览器，登录后自动保存 cookies.json，再通过「上传文件」上传
