# Admin Product Terms

**Why**：术语统一是 UX 一致性的基础。

## Rule

- `ProfileGroup` 应显示为 `Segment`
- Segment detail page 包含 individual Profile
- Segment 与 Profile 管理应在 frontend prototype 支持：create / delete / search-read /
  update / import / LLM generation 流程
- Budget ceiling input 应**直接接受数字**，不是固定 select list
