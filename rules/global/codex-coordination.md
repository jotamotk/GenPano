# Codex Coordination Workflow

**Why**：速度 + 可回溯。issue 应让下一个 session 一眼看清核心问题、当前判断、下一步动作。
不要造没改善执行的 process artifact。

## Rule

Genpano 工作通过 GitHub 协调：

- GitHub issue / PR / review comment / CI 跑 = **durable collaboration state**
- chat 仅供讨论和澄清；**不替代 issue / PR / review 状态**

issue 应该让下一个 Codex session 立刻明白：核心问题、当前判断、下一步动作。
