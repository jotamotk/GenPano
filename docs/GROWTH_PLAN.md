**内容3: 《GEO 与 SEO 的本质区别》长文**

**论点框架**:
- SEO 已是过时技能，GEO 是 2026 年新赛道
- 核心区别: 搜索引擎 vs 推荐引擎，关键词 vs 语义理解，排名 vs 推荐位置
- GEO 策略应该如何调整 (从链接权重 → 内容语义 → 品牌声誉)

**文章结构** (4000字):
```
第一部分: 为什么 SEO 不够了 (1000字)
- SEO 的成功案例: 2000-2020 年十年红利
- AI 搜索的冲击: Google 流量下降预测，AI 查询超越传统搜索
- 传统 SEO 工具在 AI 时代的局限
- 数据对标: Semrush 小样本 vs GENPANO 大样本

第二部分: GEO vs SEO - 五个核心差异 (2000字)
  差异1: 查询类型不同
    - SEO: 结构化 + 长尾关键词 (平均 3-5 词)
    - GEO: 自然语言 + 多轮对话 (整句整段)
    - 实例对比
  
  差异2: 排名机制不同
    - SEO: 基于链接权重、技术 SEO、内容相关度
    - GEO: 基于 LLM 的语义理解、训练数据 + 推荐模型
    - AI 如何评估品牌可信度 (非超链接)
  
  差异3: 优化方向不同
    - SEO: 优化网页元素、建立外链、抢占关键词
    - GEO: 优化品牌声誉、内容语义质量、媒体提及
    - 例如: 一篇权威媒体报道对 GEO 的影响 vs SEO
  
  差异4: 目标受众不同
    - SEO: 主要是搜索意图明确的用户 (已进入考虑阶段)
    - GEO: 包含探索、推荐、发现意图 (更早阶段触及)
  
  差异5: 竞争格局不同
    - SEO: 成熟、标准化、卷
    - GEO: 新兴、蓝海、品牌差异化空间大

第三部分: GEO 时代的新策略 (800字)
- 策略1: 从关键词思维 → 话题思维
- 策略2: 从权重思维 → 声誉思维
- 策略3: 从单点优化 → 全渠道协同
- 实例: 某品牌如何通过 GEO 思维反转在 AI 中的评价

第四部分: 如何用 GENPANO 制定 GEO 计划 (200字)
```

**发布策略**:
- 知乎专栏发布完整版 (获赞和评论是重点)
- 微信公众号作为主要分发渠道 (预留为粉丝专属)

## 3. 产品内病毒传播机制 (Product-Built Viral Mechanisms)

### 3.1 品牌 GEO 体检报告 PDF (Brand GEO Health Report)

> **产品定位**: 不是社交截图，而是**专业 PDF 体检报告**。用户可下载发给老板、客户、同事，或作为 B2B 内容营销素材。对标 Semrush Domain Overview PDF。详细结构见 PRD 4.6.3 功能 1。

#### 3.1.1 报告格式

- **格式**: PDF, A4 纵向, 4-6 页
- **公开页**: `/brand-report/:brandId?locale=zh-CN|en-US` (无需登录可访问，SEO 索引)
- **下载**: 公开页 "下载 PDF" 按钮 → `GET /api/v1/brands/:id/share-report.pdf`
- **社交预览**: 公开页带 Open Graph 图 (1200×630，自动生成，含品牌名 + PANO Score)，贴到微信/LinkedIn 时有好看预览卡

**6 页结构摘要** (详见 PRD 4.6.3):
1. 封面 — 品牌名 + 大号 PANO + 等级 + 一句话结论 + 二维码
2. 总览与子维度 — 四维雷达 + 30 天趋势 + 3 条关键发现
3. 引擎分解 — ChatGPT / 豆包 / DeepSeek 对比
4. 竞品对标 — 本品牌 + Top 4 竞品
5. 诊断摘要 — Top 3 P0/P1 诊断 (不含执行步骤)
6. 关于 & CTA — 方法说明 + 注册/咨询入口 + 报告唯一 ID

#### 3.1.2 分享渠道和触发点

**触发点1: Dashboard 品牌 PANO 卡片**
- 品牌 PANO Score 卡片右上角"分享报告"按钮 → 新标签打开公开页 `/brand-report/:brandId`
- 公开页提供:
  - "下载 PDF" (主 CTA, 直接触发浏览器下载)
  - "复制分享链接" (次 CTA, URL + 自动生成的社交文案)
  - 社交文案示例:
    - zh: "我刚做了一份 [品牌] 的 AI 搜索 GEO 体检 - PANO Score 76 分。完整报告:{url}"
    - en: "Just generated a GEO health report for [Brand] - PANO Score 76. Full PDF: {url}"

**触发点2: 分数提升时**
- 如果 PANO Score 对比上周提升 5+，Dashboard 顶部弹出"分享你的进步"banner
- 指向"分享报告"按钮，带预设分享文案

**触发点3: 排名变化时**
- 品牌进入行业 Top 3 时，自动生成"庆贺"弹窗
- 附带 CTA: "下载报告 PDF 发给老板"

**触发点4: 邀请新用户**
- Dashboard 中的"邀请同事/客户"功能
- 复制邀请链接时自动附带邀请人品牌的 `/brand-report/:id` 链接
- 邀请人获得积分奖励

**B2B 场景增强**:
- 用户可在公开页选择"加密 PDF" (选填密码), 适合客户提案场景
- PDF 内嵌报告唯一 ID (格式 `RPT-{industryCode}-{YYYYMMDD}-{random6}`), 便于追踪分享来源

#### 3.1.3 预期转化率

- **PDF 下载率**: 8-12% (日活用户中 — 比 PNG 低一些，因为 PDF 是更严肃的行动)
- **但下载 → 注册转化率高**: 5-8% (因为下载 PDF 的用户大概率是 B2B 决策者，比随手发朋友圈的用户价值更高)
- **阅读 PDF 的第二度传播**: PDF 往往会转给 2-3 个同事/客户 (相当于 2-3x 放大)
- 预期月度新增: 60-120 个用户通过 PDF 报告传播 (5000 DAU × 10% 下载率 × 6% 注册转化 × 2.5x 第二度传播放大 ≈ 75 个/月, 保守 60-120 区间)
- **额外价值**: 每份带 ID 的 PDF 都是一次品牌曝光, 即使读者当下不注册也能形成品牌认知资产
